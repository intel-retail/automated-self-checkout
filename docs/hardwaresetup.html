Download Ubuntu 22.04 and follow these (installation steps](https://ubuntu.com/tutorials/install-ubuntu-desktop#1-overview).

Step 2: Update the Kernel

Note: After the kernel is updated, apt-get install might not work due to the unsupported kernel versions that were installed. To resolve this issue, do the following:

        # Find all installed kernels 
        dpkg --list | grep -E -i --color 'linux-image|linux-headers'

        # Then remove the unsupported kernels, the below example will remove the installed kernel 5.19:
        sudo apt-get purge -f 'linux--5.19'
